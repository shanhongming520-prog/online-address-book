## 一、JDK 17 下载安装

### ⚠️ 重要提示
项目基于 **Spring Boot 3.2.5**，**必须使用 JDK 17 或更高版本**。当前系统只有 JDK 8，无法编译和运行。

### 下载链接（国内镜像优先）

| 版本 | 链接 | 推荐度 |
|------|------|--------|
| **阿里镜像 JDK 17** | https://mirrors.aliyun.com/adoptium-jdk/17/jdk/x64/windows/OpenJDK17U-jdk_x64_windows_hotspot_17.0.16_8.zip | ⭐⭐⭐⭐⭐ 最快 |
| 华为镜像 JDK 17 | https://repo.huaweicloud.com/java/jdk/17.0.2+8/jdk-17.0.2_windows-x64_bin.zip | ⭐⭐⭐⭐ |
| Oracle JDK 17 (需登录) | https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html | ⭐⭐⭐ |
| Microsoft Build of OpenJDK | https://aka.ms/download-jdk/microsoft-jdk-17.0.16-windows-x64.msi | ⭐⭐⭐ |
| Eclipse Adoptium | https://adoptium.net/temurin/releases/?version=17 | ⭐⭐⭐⭐ |

### 安装步骤（以阿里镜像 ZIP 为例）

```powershell
# 1. 下载（在浏览器中打开链接下载，或用下载工具）
# 推荐文件名: OpenJDK17.zip

# 2. 解压到 E:\java17
#    解压后目录结构应为: E:\java17\bin\java.exe

# 3. 配置环境变量
#    - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
#    - 新建/修改系统变量 JAVA_HOME = E:\java17
#    - 在 Path 变量中，将 %JAVA_HOME%\bin 移到最前面
#    - 删除或修改原有指向 JDK 8 的 Java 路径

# 4. 验证
#    重新打开 CMD，运行:
java -version
#    应显示: openjdk version "17.0.16" ...
```

### 快速配置 JAVA_HOME（PowerShell 管理员模式）

```powershell
# 以管理员身份运行 PowerShell
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "E:\java17", "Machine")
$path = [System.Environment]::GetEnvironmentVariable("Path", "Machine")
# 确保 %JAVA_HOME%\bin 在最前面
$newPath = "%JAVA_HOME%\bin;" + $path
[System.Environment]::SetEnvironmentVariable("Path", $newPath, "Machine")
```

## 二、自建 Git 服务器（10.211.55.20 CentOS 7）

### 2.1 测试服务器连通性

在本地 Windows 机器上执行：

```cmd
# 方式 1: CMD
ping 10.211.55.20

# 方式 2: PowerShell
Test-Connection -ComputerName 10.211.55.20 -Count 2
```

### 2.2 在 CentOS 7 服务器上架设 Git 服务器

**SSH 登录服务器：**

```bash
ssh root@10.211.55.20
# 输入密码
```

**一键安装脚本（在 CentOS 7 上执行）：**

```bash
#!/bin/bash
# ============ Git 服务器一键安装脚本 ============
# 在 CentOS 7 上执行: bash setup-git-server.sh

echo "=== 开始配置 Git 服务器 ==="

# 1. 安装 Git
echo "[1/6] 安装 Git..."
yum install -y git

# 2. 创建 Git 用户
echo "[2/6] 创建 git 用户..."
useradd -m -s /bin/bash git
passwd git  # 交互式设置密码

# 3. 创建 Git 仓库目录
echo "[3/6] 创建仓库目录..."
mkdir -p /home/git/repositories
chown git:git /home/git/repositories
chmod 755 /home/git/repositories

# 4. 初始化裸仓库（bare repository）
echo "[4/6] 创建项目裸仓库..."
cd /home/git/repositories
git init --bare online-address-book.git
chown -R git:git /home/git/repositories/online-address-book.git

# 5. 配置 SSH 访问
echo "[5/6] 配置 SSH..."
mkdir -p /home/git/.ssh
touch /home/git/.ssh/authorized_keys
chmod 700 /home/git/.ssh
chmod 600 /home/git/.ssh/authorized_keys
chown -R git:git /home/git/.ssh

# 将本机生成的 SSH 公钥添加到 authorized_keys
# cat /path/to/your/id_rsa.pub >> /home/git/.ssh/authorized_keys

# 6. 禁用 Git 用户的 shell 登录（安全）
echo "[6/6] 配置 Git Shell 访问..."
echo "/usr/bin/git-shell" >> /etc/shells
usermod -s /usr/bin/git-shell git

# 完成
echo ""
echo "=== Git 服务器配置完成 ==="
echo ""
echo "仓库地址: ssh://git@10.211.55.20/home/git/repositories/online-address-book.git"
echo ""
echo "本地推送命令:"
echo '  git remote add private ssh://git@10.211.55.20/home/git/repositories/online-address-book.git'
echo "  git push -u private main"
```

### 2.3 本地推送代码到私有 Git 服务器

```bash
# 1. 生成 SSH 密钥（已完成，在 ssh-keys/ 目录下）
#    私钥: E:\py\1\contact-system\ssh-keys\id_rsa
#    公钥: E:\py\1\contact-system\ssh-keys\id_rsa.pub

# 2. 将公钥内容添加到服务器的 /home/git/.ssh/authorized_keys
#    使用 WinSCP 或其他 SCP 工具上传，或执行:
#    type E:\py\1\contact-system\ssh-keys\id_rsa.pub | ssh git@10.211.55.20 "cat >> ~/.ssh/authorized_keys"

# 3. 配置 SSH 使用指定私钥
#    在 E:\py\1\contact-system\ssh-keys\ 下创建 config 文件:
```

**SSH Config 文件（`E:\py\1\contact-system\ssh-keys\config`）：**

```
Host 10.211.55.20
    User git
    IdentityFile E:\py\1\contact-system\ssh-keys\id_rsa
    StrictHostKeyChecking no
```

```bash
# 4. 添加远程仓库并推送
cd E:\py\1\contact-system
git remote add private ssh://git@10.211.55.20/home/git/repositories/online-address-book.git
git push -u private main
```

### 2.4 使用 Gitosis/GitLab（可选高级方案）

如果需要更完善的管理功能（Web 界面、权限管理、CI/CD），建议安装：

- **Gitea**（轻量级，推荐）: 单二进制文件，资源占用少
- **GitLab CE**（功能最全）: 需要较多资源（至少 4GB 内存）

**Gitea 一键安装（CentOS 7）：**

```bash
# 下载 Gitea
wget -O /usr/local/bin/gitea https://dl.gitea.com/gitea/1.21/gitea-1.21-linux-amd64
chmod +x /usr/local/bin/gitea

# 创建用户和目录
useradd -m -s /bin/bash git
mkdir -p /home/git/gitea/{custom,data,log}
chown -R git:git /home/git/gitea

# 运行 Gitea（首次访问 http://10.211.55.20:3000 完成初始化配置）
su - git -c "gitea web --config /home/git/gitea/custom/conf/app.ini &"
```
