# 添加入站规则：端口 8080，协议 TCP，授权对象 0.0.0.0/0

## 🔧 快速环境安装脚本

创建文件 `install-env.sh` 并执行：

```bash
#!/bin/bash
# ============================================================
# 一键安装 Java 17 + MySQL 8.0 + Git
# ============================================================

echo "开始安装运行环境..."

# 1. 安装 Java 17
echo "[1/3] 安装 Java 17..."
yum install -y java-17-openjdk java-17-openjdk-devel

# 2. 安装 MySQL 8.0
echo "[2/3] 安装 MySQL 8.0..."
yum install -y mysql-server mysql
systemctl start mysqld
systemctl enable mysqld

# 获取临时密码
TEMP_PASS=$(grep 'temporary password' /var/log/mysqld.log | awk '{print $NF}')
echo "MySQL 临时密码：$TEMP_PASS"

# 3. 安装 Git
echo "[3/3] 安装 Git..."
yum install -y git

# 4. 配置防火墙
echo "配置防火墙..."
firewall-cmd --permanent --add-port=8080/tcp
firewall-cmd --permanent --add-port=3306/tcp
firewall-cmd --reload

echo ""
echo "=========================================="
echo "环境安装完成！"
echo "Java 版本：$(java -version 2>&1 | head -n1)"
echo "MySQL 版本：$(mysql --version)"
echo "Git 版本：$(git --version)"
echo "=========================================="
```

## 📊 部署验证

部署完成后，执行以下验证：

```bash
# 1. 检查 Java 版本
java -version
# 应显示：openjdk version "17.x.x"

# 2. 检查 MySQL 状态
systemctl status mysqld
# 应显示：active (running)

# 3. 检查应用进程
ps -ef | grep contact-system
# 应显示 Java 进程

# 4. 检查端口监听
netstat -tuln | grep :8080
# 应显示：0.0.0.0:8080

# 5. 测试访问
curl http://localhost:8080
# 应返回 HTML 内容

# 6. 测试公网访问
# 在浏览器访问：http://你的公网IP:8080
```
